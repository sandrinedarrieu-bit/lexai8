Connexion Supabase
Comment obtenir ces clés :
1. Allez sur supabase.com → créez un projet
2. Settings → API → copiez "Project URL" et "anon public"
3. Collez-les ci-dessus et cliquez "Tester"
Créer la table clients dans Supabase :
Allez dans SQL Editor et exécutez :
CREATE TABLE clients ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, type text DEFAULT 'societe', raison_sociale text, prenom text, nom text, siret text, forme_juridique text, adresse text, code_postal text, ville text, email text, telephone text, representant text, created_at timestamptz DEFAULT now() ); ALTER TABLE clients ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow all" ON clients FOR ALL USING (true);
